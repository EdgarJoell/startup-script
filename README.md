# startup-script
## Purpose
This repo is a passion project for me. Whenever it's time to get my local machine setup, I want this repo to serve as a ways of getting my machine ready for development and automatically setup, I definitely do not want to go through the hassle of going through the motions of getting everything ready when I could simply run this script and then everything would be ready.

## What Does the Script Include?
It includes but not limited to...

Languages: TypeScript, Java, Python, Ruby, etc...

Frameworks: Angular CLI, expo-cli, create-next-app, etc...

IDE's: IntelliJ Community & Premium, WebStorm, Visual Studio Code, RubyMine, Rider, etc...

Dev Tools: Node Package Manager, Node Version Manager, Java Version Manager, Homebrew, Git, etc...

## Where Did This Come From? 
I got this idea from when I worked with CVS Health where they made the onboarding process super easy and to simply run this kind of script and then bam, ready to go (at least for the most part). 

They're an enterprise establishment so obviously there were some quirks and stuff but the inital setup was really easy.
